# ResuMarkdown: Dynamic Markdown Resume Generator

An ATS-friendly resume creator designed for students to write simple plain Markdown on the left, rendering clean professional documents on the right.

## 🚀 Features
- **Live Preview Split-Pane**: Keeps inputs and styling previews side-by-side in real time.
- **Three Unique Templates**:
  - *Classic Professional*: Serif typography, traditional margins, classic look.
  - *Modern Tech*: Sans-serif Outfit layouts, distinct section fills.
  - *Elegant Minimalist*: Low spacing, upper-case headings, clean structure.
- **ATS Compliant**: Generates pure semantic clean HTML without nested canvas elements to maximize ATS scanners parsing.
- **Print Optimization**: Specific `@media print` rules hide sidebars and controls to produce clean, multi-page layout PDFs.
- **Draft Persistency**: Keeps your edits saved on reloading.

## 🛠️ Usage
1. Open the application.
2. Edit sections or select a preset dummy layout.
3. Choose a styling theme from the toolbar.
4. Click **Export to PDF** to open the print wizard. Set margins to "None" or "Default" in printer configurations.
