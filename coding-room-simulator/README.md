# Collaborative Coding Room Simulator 💻👨‍🏫

A comprehensive, simulated technical interview environment built entirely on the client side using pure HTML, CSS, and Vanilla JavaScript. Designed specifically for students preparing for software engineering interviews.

## 🌟 Features

1. **Mock Code Editor**: A syntax-highlighting-inspired code workspace where users can write, edit, and experiment with algorithmic solutions to classic programming problems.
2. **AI Interviewer Simulation**: A dynamic chat interface that simulates interaction with a technical interviewer. It greets the user, explains the problem, and dynamically provides feedback based on whether the code passes or fails test cases.
3. **Local Test Execution Engine**: A safe, client-side code runner built using JavaScript's `Function` constructor. It intercepts the user's code, injects predefined unit tests (input parameters), and asserts the expected output.
4. **Time Limits**: A built-in countdown timer that adds realistic interview pressure, stopping the session when time expires.
5. **Preloaded Prompts**: Built-in algorithmic challenges (e.g., "Two Sum", "Reverse a String", "Palindrome Checker") configured with edge cases and hidden tests.

## 🚀 Usage

1. Open the simulator from the main dashboard.
2. Use the dropdown in the sidebar to select a coding challenge.
3. The AI Interviewer will greet you and the timer will start automatically.
4. Read the problem description and begin writing your logic inside the provided boilerplate function. Make sure your function **returns** the final answer.
5. Click **"Run Code"** to execute your logic against the mock test cases.
6. Check the Console Output at the bottom right for passed/failed tests, and check the Interviewer Chat for customized feedback on your performance!
