# Interactive CSS Grid & Flexbox Layout Builder 🎨

A robust, visual development tool built purely with HTML, CSS, and vanilla JS to help developers and students quickly prototype and generate responsive layouts using Modern CSS.

## 🌟 Features

1. **Dual Modes**: Seamlessly toggle between generating a `Flexbox` layout and a `CSS Grid` layout.
2. **Interactive Controls**: Side-panel dropdowns and inputs to manipulate CSS properties dynamically:
   - **Flex**: Toggle `flex-direction`, `justify-content`, `align-items`, `flex-wrap`, and `gap`.
   - **Grid**: Define custom `grid-template-columns` (e.g., `1fr 2fr 1fr`), `grid-template-rows`, `justify-items`, `align-items`, and `gap`.
3. **Live Visual Canvas**: Instantly watch the layout container and its child items adjust as you tweak CSS properties in the sidebar. Add or remove child elements to see how the layout handles dynamic content.
4. **Live Code Export**: A dedicated bottom pane that continuously generates the exact HTML markup and CSS styling for your current visual layout, equipped with a "Copy Code" button for instantaneous use in your real projects.

## 🚀 Usage

1. **Select a Layout System**: Open the app and choose either Flexbox or Grid from the top toggle.
2. **Add Content**: Click the "Add Item" button in the canvas header to populate the container with `div` blocks.
3. **Tweak Properties**: Use the Sidebar Controls. For example, in Flexbox mode, change `justify-content` to `space-between` and watch the blocks spread out!
4. **Export Code**: Happy with the design? Scroll down to the Code Export panel and copy the ready-to-use HTML and CSS right into your own stylesheet.
