# PostLite: Browser-Based API Tester (Mini-Postman)

A lightweight client-side API testing dashboard built directly inside the browser. Helping students understand HTTP requests, analyze server responses, and copy fetch integrations without heavy desktop installations.

## 🚀 Features
- **HTTP Method Support**: Test query parameters across `GET`, `POST`, `PUT`, and `DELETE`.
- **Request Configuration Panels**: Add custom headers via key-value input fields and define raw JSON body payloads.
- **Detailed Response Indicators**: Tracks response millisecond timings and color-codes HTTP response status codes.
- **Pretty-Print JSON View**: Colorized formatting displays response payload structures cleanly.
- **Action History Sidebar**: Automatically caches the last 10 successful requests to local storage for quick reload.
- **JS Fetch Exporter**: Instantly generates equivalent JavaScript integration code snippets.

## 🛠️ Usage
1. Enter your target **Endpoint URL**.
2. Select your HTTP **Method** and configure custom headers or body payloads.
3. Click **Send** to launch. Check responses in the output panel.
4. Click the **Export JS Fetch** tab to copy the code template for your projects.
