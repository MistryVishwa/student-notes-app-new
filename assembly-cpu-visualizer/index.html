Quick Assembly Reference

MOV dest, src Copy source to destination (reg/value/address)
ADD dest, src Add source value to destination register
SUB dest, src Subtract source value from destination register
CMP op1, op2 Compare operands to set ZF and SF flags
JMP label Unconditionally branch to line label
JE label Jump to label if comparison was equal (ZF=1)
JNE label Jump to label if comparison was not equal (ZF=0)
